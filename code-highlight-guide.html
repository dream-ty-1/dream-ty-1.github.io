代码高亮配置指南

Hexo与ParticleX主题代码高亮

ParticleX主题自带代码高亮功能，需要禁用Hexo自带的代码高亮，避免冲突导致性能问题。

1. 正确的Hexo配置（_config.yml）

对于Hexo 7.0.0及以上版本，应该使用以下配置：

# 代码高亮配置 - 禁用Hexo自带高亮
syntax_highlighter:

对于较早版本的Hexo，使用以下配置：

highlight:
  enable: false
prismjs:
  enable: false

2. ParticleX主题配置（_config.particlex.yml）

ParticleX主题配置中启用代码高亮：

# 代码高亮
highlight:
  enable: true
  style: github-dark  # 可选的主题样式

实际效果

以下是几种编程语言的代码块示例：

  return `Hello, ${name}!`;

Python

# Python示例
def factorial(n):
    """计算阶乘"""
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n - 1)
    
result = factorial(5)
print(f"5的阶乘是: {result}")

性能优化

为了避免大量代码块导致页面加载卡顿，我们对代码高亮脚本进行了以下优化：

  1. 使用CSS预先定义样式，减少JavaScript处理
  2. 延迟处理代码块，等待页面主要内容渲染完成
  3. 减少DOM操作，使用DocumentFragment
  4. 避免重复处理已经增强过的代码块
  5. 使用requestIdleCallback在浏览器空闲时处理代码高亮

这些优化措施可以显著提高博客的加载速度和响应性能，尤其是在包含大量代码块的页面上。
